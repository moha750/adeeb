# صفحة روابط أدِيب 🔗

صفحة روابط متكاملة لنادي أدِيب الأدبي مشابهة لـ Linktree، مع تصميم عصري ومتجاوب.

## 📋 المحتويات

- `index.html` - الملف الرئيسي للصفحة
- `style.css` - ملف التنسيقات والتصميم
- `script.js` - ملف JavaScript للوظائف التفاعلية

## ✨ المميزات

### 🎨 التصميم
- تصميم عصري ونظيف متناسب مع هوية أدِيب
- متجاوب بالكامل مع جميع أحجام الشاشات
- دعم الوضع الليلي/النهاري مع حفظ التفضيلات
- تأثيرات حركية سلسة وجذابة

### 🔗 الروابط المتوفرة
- الموقع الإلكتروني الرسمي
- قناة يوتيوب
- حساب تويتر/X
- حساب إنستغرام
- حساب لينكد إن

### 🚀 الوظائف التفاعلية
- **عداد الزوار**: يحفظ عدد زوار الصفحة في المتصفح
- **زر المشاركة**: مشاركة الروابط عبر Web Share API أو نسخ الرابط
- **الوضع الليلي**: تبديل سلس بين الوضع النهاري والليلي
- **معاينة سريعة**: تلميحات تظهر عند المرور على الروابط
- **تأثيرات الحركة**: تأثيرات موجية وحركية عند التفاعل

### ⌨️ اختصارات لوحة المفاتيح
- `Ctrl/Cmd + K` - فتح زر المشاركة
- `Ctrl/Cmd + D` - تبديل الوضع الليلي

## 🎯 كيفية الاستخدام

1. افتح ملف `index.html` في المتصفح
2. تأكد من وجود ملفات `style.css` و `script.js` في نفس المجلد
3. تأكد من وجود لوغو أدِيب في المسار: `../adeeb-logo.png`

## 🔧 التخصيص

### تغيير الروابط
افتح ملف `index.html` وعدّل روابط `href` في عناصر `<a>`:

```html
<a href="رابطك_هنا" target="_blank" rel="noopener noreferrer" class="link-card">
```

### تغيير الألوان
افتح ملف `style.css` وعدّل المتغيرات في `:root`:

```css
:root {
    --primary-color: #2c5f2d;
    --primary-light: #97ce4c;
    /* ... */
}
```

### إضافة رابط جديد
انسخ أحد عناصر `.link-card` في HTML وعدّل:
- الأيقونة (`<i class="fab fa-...">`)
- العنوان والوصف
- الرابط
- اللون في CSS

## 🎨 الألوان المستخدمة

| اللون | الكود | الاستخدام |
|-------|------|----------|
| أخضر داكن | `#2c5f2d` | اللون الأساسي |

## 📊 الإحصائيات المحفوظة

يتم حفظ البيانات التالية في `localStorage`:
- `adeeb-theme` - الوضع الليلي/النهاري
- `adeeb-visitor-count` - عدد الزوار
- `adeeb-link-clicks` - إحصائيات النقرات على الروابط

## 🔒 الأمان

- استخدام `rel="noopener noreferrer"` لجميع الروابط الخارجية
- لا يتم جمع أي بيانات شخصية
- جميع البيانات محفوظة محلياً في المتصفح

## 📝 ملاحظات

- جميع الأيقونات من Font Awesome (CDN)
