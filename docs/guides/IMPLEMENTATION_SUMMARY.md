## 🗄️ بنية قاعدة البيانات

### الجداول المضافة/المحدثة

#### 1. `members` (محدّث)
```sql
- user_id (UUID) - ربط مع Supabase Auth
- account_status (TEXT) - حالة الحساب
- last_login (TIMESTAMPTZ) - آخر تسجيل دخول
- account_activated_at (TIMESTAMPTZ) - تاريخ التفعيل
```

#### 2. `member_invitations` (جديد)
```sql
- id (UUID)
- member_id (UUID)
- email (TEXT)
- invitation_token (TEXT)
- expires_at (TIMESTAMPTZ)
- status (TEXT)
- created_at (TIMESTAMPTZ)
- accepted_at (TIMESTAMPTZ)
- created_by (UUID)
```

#### 3. `member_activity_log` (جديد)
```sql
- id (UUID)
- member_id (UUID)
- user_id (UUID)
- activity_type (TEXT)
- activity_details (JSONB)
- ip_address (INET)
- user_agent (TEXT)
- created_at (TIMESTAMPTZ)
```

#### 4. `member_statistics` (جديد)
```sql
- id (UUID)
- member_id (UUID)
- total_logins (INTEGER)
- total_events_attended (INTEGER)
- total_certificates (INTEGER)
- profile_completion_percentage (INTEGER)
- last_updated (TIMESTAMPTZ)
```

### الدوال المضافة

1. `is_invitation_valid(token)` - التحقق من صلاحية الدعوة
2. `expire_old_invitations()` - تحديث الدعوات المنتهية
3. `log_member_activity()` - تسجيل نشاط العضو
4. `calculate_profile_completion()` - حساب نسبة اكتمال الملف
5. `update_member_last_login()` - تحديث آخر تسجيل دخول (Trigger)

## 🚀 خطوات التشغيل

### 1. إعداد قاعدة البيانات

```bash
# افتح Supabase SQL Editor
# انسخ والصق محتوى sql/member_accounts_setup.sql
# اضغط Run
```

### 2. التحقق من التثبيت

```sql
-- التحقق من الجداول
SELECT table_name 
FROM information_schema.tables 
WHERE table_schema = 'public' 
AND table_name LIKE 'member%';

-- يجب أن تظهر:
-- ✓ members
-- ✓ member_invitations
-- ✓ member_activity_log
-- ✓ member_statistics
```

### 3. اختبار النظام

```sql
-- إنشاء دعوة تجريبية
INSERT INTO member_invitations (
  member_id,
  email,
  invitation_token,
  expires_at
) 
SELECT 
  id,
  email,
  gen_random_uuid()::text,
  NOW() + INTERVAL '7 days'
FROM members 
WHERE email = 'YOUR_EMAIL@example.com';

-- الحصول على رابط التفعيل
SELECT 
  'http://localhost/members/activate.html?token=' || invitation_token
FROM member_invitations
WHERE email = 'YOUR_EMAIL@example.com'
AND status = 'pending'
ORDER BY created_at DESC
LIMIT 1;
```

### 4. تفعيل الحساب

1. افتح الرابط الناتج
2. أنشئ كلمة مرور
3. فعّل الحساب
4. سجل دخول

## 🔄 سير العمل

### للعضو الجديد

```
1. الإدارة تضيف العضو → جدول members
2. الإدارة ترسل دعوة → جدول member_invitations
3. العضو يفتح الرابط → activate.html
4. العضو ينشئ كلمة مرور → Supabase Auth
5. ربط user_id بالعضو → members.user_id
6. تحديث حالة الحساب → account_status = 'active'
7. العضو يسجل دخول → login.html
8. توجيه للوحة التحكم → dashboard.html
```

### لتسجيل الدخول

```
1. المستخدم يدخل البريد وكلمة المرور
2. Supabase Auth يتحقق من البيانات
3. النظام يتحقق من نوع الحساب:
   - إداري؟ → admin/admin.html
   - عضو؟ → members/dashboard.html
   - غير موجود؟ → رسالة خطأ
4. تسجيل النشاط في member_activity_log
5. تحديث last_login
```
