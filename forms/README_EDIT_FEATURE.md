# ميزة تعديل الاستبيانات

## نظرة عامة
تم إضافة إمكانية تعديل الاستبيانات حتى بعد نشرها. يمكن للمسؤولين الآن تحديث عنوان الاستبيان، الوصف، الأسئلة، والإعدادات في أي وقت.

## كيفية الاستخدام

### 1. الوصول إلى وضع التعديل
من لوحة التحكم (`/admin/admin.html`):
- انتقل إلى قسم "الاستبيانات"
- اضغط على زر **"تعديل"** بجانب الاستبيان المراد تعديله
- سيتم فتح صفحة المُنشئ في وضع التعديل

### 2. إجراء التعديلات
في صفحة التعديل يمكنك:
- ✏️ تغيير عنوان الاستبيان
- 📝 تحديث الوصف
- ➕ إضافة أسئلة جديدة
- ✂️ حذف أسئلة موجودة
- 🔄 تعديل نص الأسئلة وخياراتها
- 🔀 إعادة ترتيب الأسئلة
- 🔒 تغيير حالة "مطلوب" للأسئلة
- 🌐 تبديل بين عام/خاص

### 3. حفظ التغييرات
- اضغط على زر **"تحديث الاستبيان"** (بدلاً من "نشر الاستبيان")
- سيتم تحديث الاستبيان مع الحفاظ على:
  - معرف الاستبيان (ID)
  - الرابط الفريد (slug)
  - جميع الردود السابقة

## التغييرات التقنية

### ملفات معدلة:

#### 1. `/admin/admin.js`
- إضافة زر "تعديل" في بطاقة كل استبيان
- معالج الحدث `edit` الذي يفتح صفحة المُنشئ مع معامل `?edit=FORM_ID`

#### 2. `/forms/builder.js`
- إضافة `editingFormId` و `hasUnsavedChanges` إلى الحالة (state)
- دالة `loadFormForEdit()` لتحميل بيانات الاستبيان الموجود
- تعديل دالة النشر لدعم التحديث والإنشاء
- التحقق من صلاحيات المستخدم قبل التعديل
- تحديث واجهة المستخدم (العنوان والأزرار) في وضع التعديل
- **تحذير عند المغادرة بدون حفظ** (`beforeunload` event)
- **تتبع التغييرات** على جميع الحقول
- **مؤشر تحميل** عند تحميل الاستبيان
- **زر العودة** إلى لوحة التحكم بعد التحديث
- **رسائل خطأ ونجاح محسّنة** مع ألوان مميزة

#### 3. `/forms/builder.html`
- إضافة `id` للعناصر `pageTitle` و `pageSubtitle` لتحديثها ديناميكياً
- **شارة وضع التعديل** (`editModeBadge`) بتصميم جذاب
- **مؤشر التحميل** (`loadingNotice`) مع أيقونة دوارة

## آلية العمل

### عند فتح رابط التعديل:
1. يتم استخراج معرف الاستبيان من URL (`?edit=FORM_ID`)
2. التحقق من تسجيل دخول المستخدم
3. تحميل بيانات الاستبيان من جدول `forms`
4. التحقق من أن المستخدم هو مالك الاستبيان
5. تحميل الأسئلة من جدول `form_questions`
6. ملء النموذج بالبيانات الموجودة
7. تغيير واجهة المستخدم لوضع التعديل

### عند الضغط على "تحديث الاستبيان":
1. التحقق من صحة البيانات
2. تحديث بيانات الاستبيان في جدول `forms`
4. إدراج الأسئلة الجديدة/المحدثة
5. عرض رسالة نجاح

## ملاحظات مهمة

### ✨ المزايا الرئيسية

- 📝 تعديل كامل للاستبيان (العنوان، الوصف، الأسئلة)
- 🔗 الحفاظ على الرابط الفريد للاستبيان
- 💾 عدم التأثير على الردود الموجودة
- 🔒 التحقق من الصلاحيات (فقط المالك)
- 🎨 واجهة مستخدم واضحة تمیز بين الإنشاء والتعديل
- ⚡ تحديث فوري للبيانات
- ⚠️ **تحذير عند المغادرة** إذا كانت هناك تغييرات غير محفوظة
- 🔄 **مؤشر تحميل** أثناء جلب بيانات الاستبيان
- 🎯 **شارة مرئية** تظهر في وضع التعديل
- ↩️ **زر العودة السريع** إلى لوحة التحكم بعد التحديث
- ✅ **رسائل واضحة** للنجاح والفشل مع ألوان مميزة
- يُنصح بعدم إجراء تغييرات جذرية بعد جمع عدد كبير من الردود

### 🔒 الأمان:
- التحقق من صلاحيات المستخدم (owner_id)
- فقط مالك الاستبيان يمكنه تعديله

## أمثلة الاستخدام

### مثال 1: تصحيح خطأ إملائي
```
1. افتح لوحة التحكم
2. اذهب إلى "الاستبيانات"
3. اضغط "تعديل" على الاستبيان المطلوب
4. صحح الخطأ في عنوان السؤال
5. اضغط "تحديث الاستبيان"
```

### مثال 2: إضافة سؤال جديد
```
1. افتح الاستبيان للتعديل
2. اختر نوع السؤال الجديد
3. اضغط "إضافة سؤال"
4. املأ تفاصيل السؤال
5. اضغط "تحديث الاستبيان"
```

### مثال 3: تغيير خيارات سؤال اختيار متعدد
```
1. افتح الاستبيان للتعديل
2. عدّل الخيارات في السؤال المطلوب
3. أضف أو احذف خيارات حسب الحاجة
4. اضغط "تحديث الاستبيان"
```
