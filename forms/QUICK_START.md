# 🚀 دليل البدء السريع - تعديل الاستبيانات

## ⚡ في 30 ثانية

### تعديل استبيان موجود
```
1. لوحة التحكم → الاستبيانات
2. اضغط "تعديل" ✏️
3. عدّل ما تريد
4. اضغط "تحديث الاستبيان" ✅
```

### نسخ استبيان
```
1. لوحة التحكم → الاستبيانات  
2. اضغط "نسخ" 📋
3. عدّل النسخة بأمان
4. انشر عندما تكون جاهزاً
```

---

## 🎯 متى تستخدم ماذا؟

### استخدم "تعديل" ✏️ عندما:
- ✅ تصحيح خطأ إملائي
- ✅ تحديث وصف بسيط
- ✅ إضافة سؤال جديد
- ✅ تعديل خيار في سؤال

### استخدم "نسخ" 📋 عندما:
- ✅ الاستبيان لديه ردود كثيرة
- ✅ تريد تغيير جذري
- ✅ حذف أسئلة موجودة
- ✅ تغيير نوع الأسئلة

---

## ⚠️ تحذيرات مهمة

### 🔴 لا تفعل:
- ❌ حذف أسئلة من استبيان لديه ردود
- ❌ تغيير نوع سؤال بعد جمع ردود
- ❌ تعديل خيارات بشكل جذري

### 🟢 افعل:
- ✅ راجع عدد الردود أولاً
- ✅ استخدم النسخ للتغييرات الكبيرة
- ✅ احفظ التغييرات بانتظام
- ✅ اختبر بعد التعديل

## 🔧 حل المشاكل السريع

### "تعذر تحميل الاستبيان"
```
✅ تحقق من تسجيل الدخول
✅ تأكد من أنك المالك
✅ راجع اتصال الإنترنت
```

### "ليس لديك صلاحية"
```
✅ هل أنت مالك الاستبيان؟
✅ هل سجلت الدخول بالحساب الصحيح؟
```

### "تعذر التحديث"
```
✅ راجع console المتصفح
✅ تحقق من Supabase
✅ أعد المحاولة
```

---

## 📋 قائمة التحقق

قبل التعديل:
- [ ] راجعت عدد الردود
- [ ] قررت: تعديل أم نسخ؟
- [ ] عرفت ما سأغيره بالضبط

أثناء التعديل:
- [ ] أجريت التغييرات المطلوبة
- [ ] راجعت التغييرات
- [ ] اختبرت الأسئلة الجديدة

بعد التعديل:
- [ ] ظهرت رسالة النجاح
- [ ] اختبرت الاستبيان
- [ ] أبلغت المستخدمين (إن لزم)

---

## 🎓 نصائح احترافية

### 💡 نصيحة 1: المعاينة
```
اضغط "رابط التعبئة" قبل التعديل
لمعاينة الاستبيان الحالي
```

### 💡 نصيحة 2: النسخ الاحتياطي
```
استخدم "نسخ" لإنشاء نسخة احتياطية
قبل التعديلات الكبيرة
```

### 💡 نصيحة 3: التدرج
```
عدّل سؤال واحد في المرة
واختبر قبل المتابعة
```
