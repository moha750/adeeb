### 1. إدارة الصلاحيات الإدارية
```
من تبويب "أعضاء النادي":
1. ابحث عن العضو
2. اضغط "إعدادات" ⚙️
3. فعّل "منح صلاحيات إدارية"
4. اختر الدور:
   - 🟡 مدير عام (super_admin)
   - 🔵 مدير (admin)
   - 🟢 مشرف (moderator)
5. (اختياري) حدد الصلاحيات التفصيلية
6. احفظ ✅
```

### 2. إدارة المجلس الإداري
```
من نفس نافذة الإعدادات:
1. فعّل "إضافة للمجلس الإداري"
2. اختر المنصب:
   - 👑 الرئيس
   - 🥈 نائب الرئيس
   - 👔 القائد
   - 🎖️ نائب القائد
   - 💼 الرئيس التنفيذي
   - 📝 الأمين العام
   - 💰 أمين الصندوق
   - 👤 عضو
3. حدد ترتيب العرض
4. (اختياري) أضف القسم/اللجنة
5. احفظ ✅
```

### 3. عرض المستخدمين الإداريين
```
من تبويب "الصلاحيات الإدارية":

- معلومات: الاسم، البريد، الدور، الحالة
- ألوان مميزة لكل دور
- أزرار: تعديل ✏️ | إزالة 🗑️

   ↓
3. يختار المنصب (رئيس / نائب / قائد / إلخ)
   ↓
4. يحدد ترتيب العرض
   ↓

   ↓

   ↓
7. يتم الحفظ في جدول board_positions
   ↓

1. اضغط زر "إزالة" 🗑️ → تأكيد → تم ✅
```
