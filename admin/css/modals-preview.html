أنواع النوافذ — اضغط على أي بطاقة لفتح النافذة (باللون المختار أعلاه)
أساسية
basic
النافذة الأساسية: header + body + footer مع عنوان فرعي اختياري.
.modal .modal-header .modal-body .modal-footer
رسالة نجاح
success-simple
نافذة نجاح بسيطة: عنوان + رسالة + زر "ممتاز!" واحد.
.modal-success fa-circle-check
رسالة خطأ
error-simple
نافذة خطأ: عنوان + صندوق خطأ box-danger + أزرار.
.modal-danger .modal-info-box.box-danger
تأكيد حذف
confirm
بدون header — أيقونة كبيرة مركزية + عنوان + رسالة + أزرار.
.modal-confirm .modal-confirm-icon .modal-confirm-title
نموذج إدخال
form
نافذة بحقول إدخال: form-row + form-group + input/select/textarea.
.form-row .form-group .form-input .form-hint
نموذج بعمودين
form-grid
شبكة حقول بعمودين مع دعم full-width + صندوق تنبيه + required-dot.
.modal-form-grid .full-width .required-dot
تفاصيل عضو
detail
صورة شخصية + شبكة تفاصيل + صفوف تفاصيل + أقسام متعددة.
.modal-media--avatar .modal-detail-grid .modal-detail-row .modal-section
مشاركة
share
شبكة أيقونات التواصل الاجتماعي + حقل نسخ الرابط.
.share-options-grid .share-option .share-link-container
قص صورة
cropper
شريط أدوات + كانفس + مقابض + أزرار نسبة + معاينة مباشرة.
.modal-cropper .cropper-toolbar .cropper-canvas-area .cropper-controls-row
